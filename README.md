# Fennix

Opeating System from scratch made in C and C++

---

#### How to compile & run

- Clone repo using:
```bash
git clone --recurse-submodules https://github.com/Fennix-Project/Fennix.git
```

- Build the cross-compiler and other tools:
```bash
make tools
```

- Compile using:
```bash
make build
```

- Run using:
```bash
make run
```

- Clean using:
```bash
make clean
```
